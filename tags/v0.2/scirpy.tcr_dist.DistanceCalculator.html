scirpy.tcr_dist.DistanceCalculator¶

class scirpy.tcr_dist.DistanceCalculator(cutoff, n_jobs=None)¶

Abstract base class for a CDR3-sequence distance calculator.

Parameters
cutoff : floatfloat

Will eleminate distances > cutoff to make efficient use of sparse matrices.

n_jobs : int, NoneOptional[int] (default: None)

Number of jobs to use for the pairwise distance calculation. If None, use all jobs.

Attributes

DTYPE

The sparse matrix dtype.

Methods

calc_dist_mat(seqs)

Calculate pairwise distance matrix of all sequences in seqs.

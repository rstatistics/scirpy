scirpy.pl.clonotype_network_igraph¶

scirpy.pl.clonotype_network_igraph(adata, neighbors_key='tcr_neighbors', basis='clonotype_network')¶

Get an igraph object representing the clonotype network.

Requires running scirpy.tl.clonotype_network() before, to compute the layout.

Parameters
adata

Annotated data matrix.

neighbors_key

Key in adata.uns where tcr neighborhood information is located.

basis

Key in adata.obsm where the network layout is stored.

Return type

Tuple[Graph, Layout]Tuple[Graph, Layout]

Returns

graphGraphGraph

igraph object

layoutLayoutLayout

corresponding igraph Layout object.
